## üõ†Ô∏è Local Installation

> Note: Ensure that the frontend project is running before proceeding with these steps. The frontend project can be found at [here](https://github.com/SBG-Corporate/deeplink-frontend).

Setting up local environment is extremely easy and straight forward. Follow the below step and you will be ready to contribute 

1. Clone the code locally using:
   ```
   git clone https://github.com/SBG-Corporate/deeplink-backend.git
   ```
2. Switch to the code folder:
   ```
   cd deeplink-backend
   ```
4. Create a `.env` file in the root directory and add the necessary environment variables as specified in `.env.template`
5. Create your feature or fix branch you plan to work on using:
   ```
   git checkout -b <feature-branch-name>
   ```
6. Install the dependecies:
   ```
   yarn
   ```
7. Start the development server:
   ```
   yarn dev
   ```

You are ready to make changes to the code. Do not forget to refresh the browser (in case it does not auto-reload).

Thats it!
